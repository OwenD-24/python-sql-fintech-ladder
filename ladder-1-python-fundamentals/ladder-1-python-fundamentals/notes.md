## Flow control

Flow control means Python chooses what code to run based on conditions.

An `if` statement checks a condition.

An `elif` checks another condition if the first one was false.

An `else` runs when none of the earlier conditions matched.

## Loops

A `while` loop keeps running while its condition is true.

`while True` keeps looping until something inside the loop stops it.

`break` exits the loop.

A `for` loop goes through each item in a list one by one.

`continue` skips the current loop round and moves to the next item.
